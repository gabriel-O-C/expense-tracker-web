# Expense Tracker

###

## First Step

### `yarn install` or `npm install`

running this command you will install the dependencies needed to run the project.

###

## Available Scripts

In the project directory, you can run:

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

### Project purpose

The main idea of the project is to be a Expense Tracker to help people to have a better controll of their finances.
